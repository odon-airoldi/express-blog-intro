## Esercizio

# Crea il progetto base
con una rotta di benvenuto che ritorna un testo semplice con scritto ”Server del mio blog”

# Creiamo un array di oggetti
inserisci una lista di almeno 5 post, per ognuno indicare
- titolo
- contenuto
- immagine
- tags (tags è un array di stringhe)

# Crea una rotta /bacheca
che restituisca un oggetto JSON con la lista dei post

# Configura gli asset statici sull’applicazione
per visualizzare le immagini associate ad ogni post
